# FinTrack — Front-end

Interface web do FinTrack, aplicação de controle de finanças pessoais. Construída em Angular com componentes standalone e signals, consumindo a API REST do back-end.

🔗 **[Ver projeto no ar](https://finance-tracker-front-theta.vercel.app)** · **[Repositório do back-end](https://github.com/ExoticAndrew/finance-tracker-api)**

> A documentação completa (arquitetura, decisões técnicas, endpoints) está no repositório do back-end.

---

## ⚙️ Tecnologias

Angular, TypeScript, SCSS, Chart.js, RxJS

---

## 🚀 Como rodar localmente

```bash
git clone https://github.com/ExoticAndrew/finance-tracker-front.git
cd finance-tracker-front
npm install
ng serve
```

Abre em `http://localhost:4200`. Requer o back-end rodando localmente (ver instruções no [repositório da API](https://github.com/ExoticAndrew/finance-tracker-api)).
